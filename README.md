# Team Design Studios — React Landing Page

A branded React application built for Team Design Studios (Eternal Graphx, LLC).

## Features

- Custom TDS Spin Network icon with cyan neon glow animation
- TDS lockup logo with static display
- Responsive centered layout
- Live link to Team Design Studios website
- Built with React, CSS animations, and custom asset imports

## Tech Stack

- React 18
- CSS3 Animations & Filter Effects
- Custom SVG/PNG asset integration
- Git version control

## About

This project was built as part of a React learning journey and portfolio development initiative for Team Design Studios, a design consultancy specializing in UX Research & Design, Brand Identity, and Creative Solutions.
